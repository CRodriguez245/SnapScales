# Hex Composition

**Challenge wheel:** wedges show **1–6** only. **Full challenges** appear in the numbered list under the wheel (read there). **Spin challenge** and **Spin win** are **separate** buttons. After both are set, **Start round** (40s).

## Run

Open `index.html` or serve this folder locally.

## Flow

1. **Spin challenge** — new set of six challenges, wheel picks 1–6; selected row is highlighted.
2. **Spin win** — Pieces vs Cover.
3. **Start round** — build, then score.

Same hex size: piece count matches area.
