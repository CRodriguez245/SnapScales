# SnapScales

A satisfying, playful gap-triggered growth game where scales snap into place tightly on a hexagonal lattice.

## Gameplay

- Click glowing sockets to place scales
- Fill every socket inside the boundary
- Cross the line and you lose
- Multiple boundaries appear progressively as you play
- Each boundary has its own unique color palette

## Features

- Hexagonal lattice-based placement system
- Progressive boundary spawning
- Curved and geometric boundary shapes
- Color-coordinated boundaries (scales, sockets, and outlines match)
- Satisfying snap animations
- Particle effects on placement

## How to Play

1. Click on any glowing socket (green circles) to place a scale
2. Scales must be placed inside the dashed boundary lines
3. Fill all available sockets within each boundary to win
4. Placing a scale outside any boundary results in a loss

## Technical Details

- Built with p5.js
- Single HTML file - no build step required
- Responsive canvas that adapts to window width
- Deterministic socket generation
- Stable per-tile randomization for consistent visuals
